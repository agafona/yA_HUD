I also wrote a FAQ including useful links & various console commands

To download the HUD press the ZIP button on the left.

To install navigate to Steam / steamapps / common / team fortress 2 / tf / custom / . There you will want to create a folder with a random name (you can call it "hud" for example) and then copy the resource and the scripts folder (from the HUD version that you want to use) into that newly created folder.

This SteamGroup will be used for updates and general communication.
