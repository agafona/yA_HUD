Screenshots: http://imgur.com/a/xylsT

-----------------------------------------------------------

To DOWNLOAD the HUD press the "ZIP" button on top of the github page.

-----------------------------------------------------------

To INSTALL, 
go to Steam / steamapps / common / team fortress 2 / tf / custom / 
create a folder and name it, for example, hud / place resource and scripts in it. 

-----------------------------------------------------------

If you're using a 16:10 or 4:3 monitor:

1. Navigate to resource/ui.
2. Change the Name of "MainMenuOverride.res" to something like "MainMenuOverride - OLD.res"
2. Rename "ScoreBoard - 16BY10.res" or "- 4BY3" to "MainMenuOverride.res"

This will fix the MainMenu for you aspect ratio.

----------------------------------------------------------

If you're using a 4:3 monitor do the same thing with the Scoreboard.res file to fix the 6v6 Scoreboard

-----------------------------------------------------------

You can also use the "TextWindow - BIGHTML" to replace "TextWindow.res" to get a bigger HTML MOTD screen (sizzling stats for example)
